# Paradise Nursery Shopping Application

A modern and responsive plant shopping web application built using React.js.
This project allows users to browse plants, view details, and manage a shopping cart.

## Features

* Browse a variety of plants
* Add items to cart
* Increase / decrease quantity
* View total price
* Responsive design
* Fast and lightweight UI

## Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5 & CSS3

## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/paradise-nursery.git
cd paradise-nursery
npm install
npm start
```
